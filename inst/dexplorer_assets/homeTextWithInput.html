DExploreR you can interactively explore your bulk RNA-seq data. To get started, select a data set from the Data sets tab. These are publicly available data sets processed using the R package DExploreR.

Alternatively, you can upload your own data set in the required format. To meet the requirements, you can either use the functions of the DExploreR R package as suggested in the vignette or use your own processing pipeline and as an additional last step create a 
dexDataSet
object using 
dexplorer::createDataSet()
. This function creates the data set based on your input and checks validity to meet the app's requirements.
